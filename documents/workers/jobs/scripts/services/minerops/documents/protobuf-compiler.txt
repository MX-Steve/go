一. protobuf-compiler
1. 概念
    是 Google 公司内部的混合语言数据标准，提供了一种轻量高效的结构化数据存储。
2. 安装
    ]# yum -y install autoconf automake libtool curl make g++ unzip
    ]# yum -y install protobuf-compiler
    ]# git clone https://github.com/protocolbuffers/protobuf.git
    ]# cd protobuf
    ]# git submodule update --init --recursive
    如果报错，执行 git init
    ]# ./autogen.sh
    ]# ./configure
    ]# make && make install
    ]# ldconfig
    ]# mv /usr/bin/protoc /usr/bin/protoc.bk
    ]# ln -s /usr/local/bin/protoc /usr/bin/protoc
二. protoc-gen-go
1. 概念
    是 protobuf 编译插件系列中的 Go 版本
2. 安装
    ]# go get -u github.com/golang/protobuf/protoc-gen-go
三. 项目构建
    ]# cd shepherd
    ]# mkdir build
    ]# cmake ..
    ]# ls
    CMakeCache.txt  CMakeFiles  cmake_install.cmake  CTestTestfile.cmake  Makefile  software
    ]# make
    ]# ls software/golang/bin
